# Sparkforge — Landing Page

A fully responsive landing page built for **Full Stack Web Development Internship – Task 1**.

🔗 **Live demo:** _add your GitHub Pages link here after deploying_

## About

A single-page marketing site for **Sparkforge**, a fictional build studio, designed with a blueprint / schematic visual identity — grid backgrounds, dimension-line dividers, and drafting-style callouts — to match the "Build Smarter" theme.

## Features

- **Header & navigation** with a working dropdown under Services, and a mobile hamburger menu
- **Hero section** with headline, tagline, and CTA button
- **Features section** — 4 cards with FontAwesome icons (Fast, Responsive, Scalable, Secure)
- **Services section** with three service panels
- **Footer** with copyright, Privacy Policy / Terms / Contact links, and business contact info
- Fully responsive across desktop, tablet, and mobile
- Google Fonts typography (Space Grotesk, Inter, JetBrains Mono)
- CSS gradients, hover effects, and smooth-scroll navigation

## Tech Stack

- HTML5
- CSS3 (Flexbox & Grid, custom properties, media queries)
- Vanilla JavaScript (mobile nav + dropdown toggle)
- [Google Fonts](https://fonts.google.com/)
- [Font Awesome](https://fontawesome.com/) (via CDN)

## Getting Started

Clone the repo and open `index.html` directly in your browser — no build step required:

```bash
git clone https://github.com/<your-username>/<repo-name>.git
cd <repo-name>
open index.html   # or double-click the file
```

## Deployment (GitHub Pages)

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to the `main` branch, root folder
4. Your site will be live at `https://<your-username>.github.io/<repo-name>/`
